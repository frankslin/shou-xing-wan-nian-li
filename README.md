## 发布说明

```
Post By 叶风扬：2025/8/9 14:11:00 [此贴子已经被作者于2025/8/26 11:22:15编辑过]

寿星天文历风羲版仅发布(单文件)网页版，由叶风扬(风羲)基于(原版)寿星天文历修改而成。

V5.09.04版

相较于原版(V5.09)，主要修改有：
将[日月食]表二中的方位角改为以正北为0°，正东为90°的通用表达方式；
将公元前2517年1月3日定为干支纪年的起点；
参考了1266楼 lulong 提供的资料，扩充了 黄帝 ~ 禹 的世系，该世系与我推算的干支纪年起点极为接近；
这是我依据网络资料整理的《古帝世系》，因资料缺乏充分考证，且多为神话传说，叙事荒诞，故仅供参考；
优化排版，使软件在跨平台环境都有较好的显示效果；
以 ⚆ 标示时节，以 ⊙ 标示中气；
补充更新一些节日、纪念日；
更新 [日月食]图 站心坐标的横坐标随朝向变化；
网格只标示当前激活坐标系。
其他诸多修改细节不一一赘述。

因使用了新版代码指令，本版须运行在2017年之后发行的浏览器上。
经过Windows(7/10)、Linux、Android、Harmony四种OS下多种浏览器测试，均可正常运行显示，都有较好的排版显示效果。
之前的浏览器请使用V5.04.06版。

通过网盘分享的文件：寿星天文历V5.09.04.htm.7z
```
